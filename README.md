A Streamlit-based web app that simulates an e-commerce store. Designed for both learning and practical use.

## Features

### For Users:
- Sign up and login with email verification and OTP
- Browse products with images, prices, categories, and stock info
- Add products to cart
- Checkout and place orders
- Persistent login using local session
## Bulit by Rao samad
### For Admin:
- Add new products
- View user login history
- Visualize daily logins with charts
- Admin access restricted to authorized email

## Technology Stack
- **Frontend:** Streamlit
- **Database:** SQLite
- **Backend Logic:** Python
- **Data Storage:** JSON files for products and images
- **Email OTP:** Gmail SMTP server
